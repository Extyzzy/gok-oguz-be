ругается:
	Could not resolve dependency:
	npm ERR! peer typeorm@"^0.2.24" from typeorm-seeding@1.6.1

решение:
	1. становка может произойти не в текущую папку, а в папку которая указанах в путях каталога ".idea", поэтому для начала можно удалить папку ".idea", для тог, чтобы установка прошла в текущую папку. 

	2. npm install --force
